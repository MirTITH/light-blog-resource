# gnome-theme

## 安装主题



```bash
cp -r .themes ~/
cp -r .icons ~/

sudo apt -y install gnome-tweaks chrome-gnome-shell 
```

然后在 gnome-tweaks 中选



**应用于qt程序(效果不好，算了)：**

> https://github.com/desktop-app/qt5ct

`sudo apt install qt5ct`

Add line 'export QT_QPA_PLATFORMTHEME=qt5ct' to ~/.profile and re-login.

然后在 qt5ct 里设置
